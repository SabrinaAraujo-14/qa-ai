## 🎯 **O que é?**

**QA.AI** é uma ferramenta que usa inteligência artificial para gerar automaticamente:

- ✅ **Casos de teste funcionais** (positivos + negativos)
- ✅ **Cenários BDD** (Given/When/Then)
- ✅ **Cobertura completa** de cenários edge-case

**Entrada**: "Usuário deve fazer login com email e senha"
**Saída**: 5+ casos de teste prontos em segundos!
